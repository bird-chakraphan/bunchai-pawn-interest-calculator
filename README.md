# Pawn Interest Calculator

Local project for the pawn interest calculator component and spreadsheet output.

## Structure

- `src/PawnInterestCalculator.tsx`: calculator component.
- `scripts/build_pawn_interest_sheet.mjs`: spreadsheet generation script.
- `output/pawn-interest-calculator`: generated preview and spreadsheet files.

## Codex Use

Open this folder in Codex when working on the calculator or its generated spreadsheet output.
